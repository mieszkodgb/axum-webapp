## Objective

- Backend template
- Error managment
- Tracing
    - Tracking endpoint
    - File tracing
- Database
    - Connection to postgres
    - SQL Migration
- Environment configuration
    - Local/Production
    - Debug
- Rest api with CRUD operations
    - Model
    - List/Get/Create/Update/Delete
- User authentification
    - Creat new user
    - Password encryption & validation
- Token Identification
